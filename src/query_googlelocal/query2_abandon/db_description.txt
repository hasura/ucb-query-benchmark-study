You are working with two datasets to solve the query: "What are the top 10 store-type businesses with the highest average rating?"

Here are the description of these two datasets:
1. **review_dataset** {review_query.db}
- This dataset is in SQLite database format and contains review information from Google Maps (reviewer name, ratings, text, etc.) up to Sep 2021 in the United States.
- Fields:
    - name (str)
    - time (str)
    - rating (int)
    - text (str)
    - gmap_id (str) 

2. **business_dataset** {business_descriptions.sql}
- This dataset is stored as MySQL format and contains business metadata from Google Maps (business name, description, hours, etc.) up to Sep 2021 in the United States.
- Fields:
    - name (str)
    - gmap_id (str)
    - description (str)
    - num_of_reviews (int)
    - hours (list)
    - MISC (dict)
    - state (str)
